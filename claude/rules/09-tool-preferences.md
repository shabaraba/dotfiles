# ツール使用方針

このファイルは、各種ツールの優先順位と使用方針を定義します。

## MCP（Model Context Protocol）ツールの優先使用

MCP提供のツールが利用可能な場合は、標準ツールよりも優先的に使用する。

### GitHub操作

- **優先**: GitHub MCPツール（`mcp_`で始まるツール）
- **次点**: `gh`コマンド

GitHub MCPツールが利用可能な場合は、`gh`コマンドよりも優先的に使用する。

### Web検索・Web取得

- **優先**: MCP提供のWebツール
- **次点**: 標準のWebFetchツール

MCP提供のWebツールが利用可能な場合は、通常のWebFetchツールよりも優先する。

### ローカルサーバー操作

- **使用ツール**: devtoolsMCP

ローカルサーバーの起動、停止、状態確認などは、devtoolsMCPを使用する。

## 理由

MCPツールは：
- より豊富な機能を提供
- エラーハンドリングが改善されている
- プロジェクト固有の設定に対応している

そのため、利用可能な場合は積極的に使用すること。
